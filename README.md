# Rock-Paper-Scissor-Simple-Game-
A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. Players can choose Rock, Paper, or Scissors, while the computer generates a random move. The game instantly displays the result (Win, Lose, or Tie) through alerts. The project also includes custom styling, hover effects, and a clean user-friendly interface.

-->Features:
1.Random computer move generation using JavaScript
2.Win/Lose/Tie result logic
3.Interactive buttons with hover effects
4.Clean and responsive UI design
5.Beginner-friendly project for learning DOM events and JavaScript functions

-->Technologies Used:
HTML
CSS
JavaScript
